Portfolio

WiFi Device Tracker

See who is nearby

This project is meant for physical store owners (or anyone else) who want to see data regarding the number of customers that come in or near the location. Using information that is collected by a scanning device, store owners can see the number of devices, as well as how long they stayed in the premises. This information can be used to infer things like number of customers, average stay length, number of people who enter but don't purchase anything, or the number of people who walk by. This project relies on AWS technologies. As such, it is completely serverless, meaning that it has the potential to be scaled to massive sizes without much effort.

Technologies:

  • Raspberry Pi with Kali Linux
  • AWS S3, Athena, Lambda
  • Serverless
  • Vue.js, Vuetify
